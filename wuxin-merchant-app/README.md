# 五鑫跑腿商家端

V1.8 商家微信小程序，使用微信原生小程序和 TypeScript 开发。

## 页面

- 登录、商家申请、审核状态
- 工作台、订单列表、订单详情
- 店铺资料、商品分类、商品管理、商品编辑
- 经营概览、个人中心

## 本地运行

```bash
npm install
npm run type-check
npm run lint
npm run build
```

在微信开发者工具中导入本目录。当前 `project.config.json` 使用
`touristappid`，发布前必须替换为正式商家端 AppID，并把
`miniprogram/config/env.ts` 的接口地址替换为已备案的 HTTPS 合法域名。

本地联调默认连接 `http://localhost:8080`。登录后根据真实商家审核状态
进入申请、审核状态或工作台页面。

## 业务边界

- 订单接单、拒单、出餐，店铺营业状态、分类和商品管理均调用真实接口。
- 经营概览只汇总真实订单分页返回的数量，不作为财务结算数据。
- 当前后端没有图片上传接口，营业执照、店铺图和商品图使用 HTTPS URL 字段。
- 不在前端自行修改订单状态，也不伪造销量、收入或退款结果。
